Making a Website

The Tools of the Trade

What We'll Be Working With

When we make a website, there are a few tools we need to have!

  • Some basic information about what we're making
  • A text editor
  • A web browser

Wireframes

When web professionals build a website, they first have to plan out what information will go in the website.

Next, they plan where to put that information in their page layout. They do this by making a wireframe, or a rough sketch of their page.

WIREFRAME TIME!

Let's take 10 minutes and draw a quick wireframe of our website.

When I make a wireframe, I like to think of my website as being kind of like a newspaper. They have a title, some headlines, some text, and some images.

Step Three: Get a Text Editor

We're going to talk about some theory, but first, we need somewhere to put our code when we write it!

Let's go to Mozilla's Thimble: https://thimble.mozilla.org/en-US/ and click on "Start a project from scratch"

Text Editors

To write HTML, we use a tool called a text editor. A text editor is a lot like Microsoft Word, or iTunes, or iMovie – it’s a piece of software that does a specific job.

A text editor usually lives on your computer, but sometimes designers and developers work in cloud-based editors. Thimble is a simple version of that.

Working in the cloud is a great option because we don’t have to download a bunch of stuff to our computers – and we can also work on our websites wherever we are, at home or at school.

Step Four: Get a Browser

This is an easy one, because you will already have it ready to go! A browser is a piece of software you install on your computer that lets you surf the web - like Chrome, Safari, Firefox, or Internet Explorer.
